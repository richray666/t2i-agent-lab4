# 提交说明

本仓库用于提交数字媒体处理技术实验四：基于预训练模型的文生图智能体。

## 当前内容

- 智能体源码：`src/t2i_agent/`
- 测试提示词：`prompts/test_prompts.jsonl`
- 运行脚本：`scripts/`
- 生成结果：`outputs/`
- 实验报告：`reports/`
- 运行手册：`RUNBOOK.md`

## 模型说明

- 国产模型：`Kwai-Kolors/Kolors-diffusers`
- 国外模型：`microsoft/Lens` / `microsoft/Lens-Turbo`
- 大模型权重和 Hugging Face 缓存不上传到 GitHub，需按运行手册在本地或服务器按需下载。

## 上传说明

初始项目通过本机 Git 推送到 GitHub；本文件用于验证 GitHub 插件已获得该仓库的写入权限。
